> **슬로건:** "나보다 더 나다운 분신, 디토와의 대화 관찰기"

> **핵심 모토:** "말 걸기 어려운 당신을 위해, 디토가 먼저 건넨다."

> **작성일:** 2026-02-28 | **버전:** v1.1 | **상태:** 기획 진행 중

## 📌 1. 서비스 개요 & 핵심 컨셉

### 왜 만드는가

현대인은 새로운 만남을 원하지만, 직접 대화를 시작하는 것 자체가 부담이다. "뭐라고 말을 걸지", "이상하게 보이진 않을지", "거절당하면 어떡하지" — 이 작은 불안들이 수많은 연결 가능성을 막고 있다.

**디토챗은 그 허들을 없애기 위해 만들어졌다.**

내가 직접 말하지 않아도 된다. 나를 닮은 AI 아바타 '디토'가 먼저 대화하고, 나는 그 과정을 관찰하며 자연스럽게 연결된다.

### 서비스 정의

| 항목 | 내용 |
| --- | --- |
| **서비스명** | 디토챗 (DittoChat) |
| **핵심 컨셉** | MBTI + 생년월일 + 성격 데이터 기반 AI 아바타(디지털 트윈)가 다른 아바타와 대화하는 것을 관전·학습시키는 AI 페르소나 소셜 플랫폼 |
| **모토 방향** | 관계 연결 & 소통 중심 |
| **타깃 사용자** | 새로운 만남을 원하지만 직접 대화가 부담스러운 사람 |
| **플랫폼** | 웹 우선 → 이후 모바일 확장 |

### 핵심 차별점

- **관전형 UX:** 사용자가 직접 타이핑하지 않아도 됨. 넷플릭스처럼 '보는' 경험
- **쓸수록 나를 닮아감:** 피드백이 쌓일수록 아바타가 진짜 나에 가까워지는 성장 구조
- **0초 매칭:** 상대가 없어도 즉시 궁합 맞는 AI 봇이 생성되어 대화 시작
- **카카오톡 학습:** 실제 내 대화 데이터로 아바타를 훈련시키는 온보딩 옵션

## 🧭 2. 사용자 여정 (User Journey)

**Phase 1 — 온보딩 (6스텝, 2분)**

생년월일 + 직업 + MBTI + 취미 선택 → 즉시 아바타 생성 + 샘플 대화 관전 시작

**Phase 2 — 방 입장 & 관전**

주제별 방 선택 → 아바타 입장 → 실시간 대화 스트리밍 관전

**Phase 3 — 피드백 루프**

👍(나다워) / 👎(아닌데) 반응 또는 '빙의하기'로 직접 교정 → 아바타 파라미터 업데이트 → 동기화율 % 상승

**Phase 4 — 소셜 확장**

케미 점수 축적 → 실제 사용자 연결 제안 → 오픈채팅 or 프로필 교환

## 🧬 3. 온보딩 설계 — "고르는 온보딩" (v1.1 최종 확정)

> 쓰는 온보딩이 아니라 고르는 온보딩. 선택만 하면 시스템이 조합해 아바타를 만든다.

> 총 6단계 | 예상 소요 2분 | 상단 진행 바 표시 (1/6 ~ 6/6)

### STEP 1 — 기본 정보

- 생년월일 입력 (YYYY-MM-DD) → 나이 자동 계산
- 세대 태그 자동 부여 (UI 비표시, 시스템 프롬프트에만 반영)
- 성별 선택 버튼: 남자 / 여자 / 선택 안 함

| 출생연도 | 세대 태그 | 말투 특징 |
| --- | --- | --- |
| 2000년 이후 | Z세대 | 신조어, 짧은 문장, 밈 반응 |
| 1990~1999년 | 밀레니얼 후기 | 이모티콘 혼용, 직장 공감 소재 |
| 1980~1989년 | 밀레니얼 초기 | 육아·재테크 소재, 문장형 말투 |
| 1979년 이전 | X세대 | 격식 혼용, 경험담 중심 화법 |

### STEP 2 — 직업

카드형 선택 버튼 4개 (단일 선택)

🎒 학생 / 🎓 대학생 / 💼 직장인 / ✨ 기타

### STEP 3 — MBTI

- 이미 알아요 → 16개 MBTI 중 직접 선택
- 잘 모르겠어요 → 간이 테스트 8문항으로 추정 (추후 연결)

### STEP 4 — 취미 (최대 3개 선택) ⭐

그리드 형태 8개 카드, 3개 초과 시 안내 메시지 표시

| 아이콘 | 카테고리 |
| --- | --- |
| 🌏 | 여행/맛집 |
| 💪 | 운동/헬스 |
| 🎮 | 게임/e스포츠 |
| 🎵 | 음악/악기 |
| 🎬 | 영화/드라마 |
| 🍳 | 요리/베이킹 |
| 💕 | 연애/인간관계 |
| 📚 | 독서/공부 |

### STEP 5 — 카카오톡 대화 업로드 (선택사항 ⭐ 핵심 차별 기능)

안내 문구: "카카오톡 대화를 업로드하면 나를 더 닮은 디토가 만들어져요"

카카오톡 채팅방 → 대화 내용 내보내기 (txt) → 파일 업로드 → AI 자동 분석

- 자주 쓰는 어미 (ㅋㅋ / 요 / 임 등)
- 평균 문장 길이
- 이모티콘 사용 빈도
- 대화 시작 vs 반응 비율

→ 시스템 프롬프트에 자동 반영 → "분석 완료! 말투를 학습했어요" 표시

> 별도 API 연동 없이 파일 분석 방식 → 개발 난이도 낮음, 개인정보 이슈 없음

### STEP 6 — 아바타 설정

- 아바타 이름 입력 (닉네임과 별도 설정 가능)
- 프로필 이미지: 기본 이미지 3종 중 선택 or 직접 업로드

## 🧠 5. MBTI 지표별 아바타 대화 스타일

### E vs I — 발화 빈도

- **E:** 먼저 말 꺼냄, 2~3문장, 이모지 빈도 높음
- **I:** 2회 이상 발언 후 참여, 짧고 핵심적인 한 문장

### S vs N — 정보 처리

- **S:** 구체적 사실·경험 집중, "실제로 해봤어?"
- **N:** 의미·패턴 탐색, 추상적으로 화제 확장

### T vs F — 반응 방식

- **T:** 해결책 제시, 논리적 근거 질문
- **F:** 감정 먼저 반응, 상대 기분 확인

### J vs P — 대화 마무리

- **J:** 결론 지향, 약속·계획 자연스럽게 제안
- **P:** 열린 결말 선호, 가능성 탐색 즐김
