# 디토챗 (DittoChat) 프로젝트 기획서

> **슬로건:** "나보다 더 나다운 분신, 디토와의 대화 관찰기"

---

## 📌 프로젝트 모토 & 만든 이유

### 우리가 디토챗을 만드는 이유

현대인은 새로운 만남을 원하지만, 직접 대화를 시작하는 것 자체가 부담이다.
"뭐라고 말을 걸지", "내가 이상하게 보이진 않을지", "거절당하면 어떡하지" —
이 작은 불안들이 수많은 연결 가능성을 막고 있다.

**디토챗은 그 허들을 없애기 위해 만들어졌다.**

내가 직접 말하지 않아도 된다.
나를 닮은 AI 아바타 '디토'가 먼저 대화하고,
나는 그 과정을 관찰하며 자연스럽게 연결된다.

> **핵심 모토:** "말 걸기 어려운 당신을 위해, 디토가 먼저 건넨다."

### 타깃 사용자
새로운 만남을 원하지만 직접 대화가 부담스러운 모든 사람.
MBTI에 관심 많은 MZ세대부터, 관계가 어렵고 피곤한 현대인까지.

## 🗺 아이디어 스트럭처: 사용자 여정 & 데이터 흐름

### 핵심 사용자 여정 (User Journey)

```
[온보딩] → [페르소나 생성] → [관찰] → [학습/보정] → [소셜 확장]
```

**Phase 1 — 온보딩 & 페르소나 생성**
사용자가 MBTI를 입력하고 객관식 선택 온보딩을 완료하면 시스템이 즉시 초기 아바타를 생성한다.
말투, 리액션 패턴, 화제 전환 빈도, 감정 표현 강도 등을 벡터값으로 초기화.
최초 1분간 '가상 메이트'와의 샘플 대화를 바로 관찰하게 해서 이탈 방지.

**Phase 2 — 방 입장 & 관찰**
사용자는 아바타를 '방'에 입장시킨다.
실시간 스트림 형식으로 대화가 진행되며, 사용자는 흘러가는 대화를 관찰한다.
넷플릭스처럼 '보는' UX가 핵심이다.

**Phase 3 — 피드백 루프**
대화 중 특정 발언에 👍(나다워) / 👎(아닌데) 반응을 남기거나,
'빙의하기'로 직접 다음 발언을 수정 입력한다.
이 데이터가 RLHF 방식으로 아바타 파라미터를 업데이트한다.

**Phase 4 — 소셜 확장**
아바타 간 '케미 점수'가 쌓이면, 실제 사용자 간 연결을 제안한다.
아바타가 '프록시(proxy)' 역할을 하여 실제 인간관계 형성의 허들을 낮춘다.

```
[사용자 입력]

        ↓
[케미 스코어 엔진]
  대화 패턴 분석 → 상성 점수 산출 → 매칭 추천
```

## ✅ MVP Must-have 기능 5가지

### 1. 페르소나 생성 엔진 (Persona Factory)
MBTI + 객관식 온보딩 데이터를 입력받아 재현 가능한 시스템 프롬프트를 생성.
아바타의 '일관성'이 서비스 신뢰의 근간. 동일 입력에 대해 매번 비슷한 말투와 반응이 나와야 사용자가 "이게 나야"라고 느낀다.

### 2. 실시간 관찰형 채팅 뷰어 (Spectator View)
WebSocket 기반 스트리밍으로 아바타 대화가 실시간 흘러가는 UI.
사용자가 직접 타이핑하지 않아도 되는 이 구조가 디토챗의 차별점.
대화 속도 조절(빠르게/느리게) 옵션도 MVP에 포함.

### 3. 가상 메이트 즉시 생성 (Instant Match Bot)
원하는 MBTI를 선택하면 0초 내에 상대 AI 봇이 생성되어 대화를 시작.
콜드 스타트 문제를 해결하는 결정적 장치.
실제 유저와 AI 봇은 시각적으로 구분하되 충분히 자연스럽게 운영.

### 4. 피드백 & 빙의 기능 (Tune My Avatar)
개별 발언에 나다워/아닌데 반응을 남기거나, '빙의하기'로 특정 턴 교정.
피드백이 쌓일수록 아바타 프로필에 '동기화율 %'로 표시 → 리텐션 루프 형성.

### 5. 방 시스템 (Room Lobby)
'육아 이야기방', 'MBTI 극상성 방', '같은 F끼리 방' 등 주제별 방 목록과 입장 기능.
MVP는 3~5개의 고정 방으로 시작. 방의 분위기와 참여 봇 수를 라벨로 표시.

## 🧬 페르소나 생성 온보딩 설계

### 핵심 방향
**"쓰는 온보딩"이 아니라 "고르는 온보딩"**
생년월일 + MBTI + 객관식 선택지 조합만으로 아바타 생성에 필요한 데이터를 충분히 확보.

---

### 온보딩 플로우 (5스텝, 예상 소요 2분)

#### STEP 1 — 기본 정보
```
생년월일 입력  →  나이 자동 계산 + 세대 태그 자동 부여
성별 선택 (남 / 여 / 선택 안 함)
```
나이는 아바타의 말투 세대감에 직접 반영.
20대 초반: "ㅋㅋ", "ㄹㅇ", 줄임말 사용 빈도 높음
40대 이상: 문장형 표현, 이모티콘 절제

#### STEP 2 — MBTI 입력
```
○ 이미 알아요  →  직접 선택 (4개 지표 토글)
○ 잘 모르겠어요  →  간이 테스트 8문항으로 추정
```

#### STEP 3 — 성격/가치관 객관식 (핵심 단계)

**① 대화 스타일**
- 🗣 먼저 말 거는 편
- 👂 듣다가 맞장구치는 편
- 💬 깊은 주제 선호
- 😄 가볍고 재밌는 대화 선호

**② 감정 표현 방식**
- 감정을 바로 표현해 / 속으로 삭히는 편
- 위로받고 싶어 / 해결책이 필요해
- 리액션 크게 하는 편 / 담담하게 반응하는 편

**③ 관심사 (다중 선택, 최대 5개)**
```
여행 / 맛집 / 독서 / 운동 / 게임 / 음악
육아 / 재테크 / 직장생활 / 연애 / 영화·드라마 / 자기계발
```

**④ 나의 관계 스타일**
- 소수 깊은 관계 선호 / 넓고 다양한 관계 선호
- 갈등 상황에서: 직접 말해 / 시간을 두고 / 그냥 넘겨

**⑤ 말투 톤 (슬라이더)**
```
진지함 ←————→ 유쾌함
격식체 ←————→ 친근체
직설적 ←————→ 돌려 말하는
```

#### STEP 4 — 나를 한 줄로 표현 (선택 조합형)
자유 텍스트 대신 조각 선택 → 자동 문장 완성 방식.
```
나는  [현실적인 / 감성적인 / 논리적인 / 엉뚱한]
      [직장인 / 학생 / 부모 / 자유로운 사람]  이고,
주로  [혼자 있을 때 / 사람들과 함께할 때]  에너지가 충전돼.
```

#### STEP 5 — 아바타 이름 & 프로필 이미지
```
아바타 이름 입력 (닉네임과 별도 가능)
프로필 이미지: AI 자동 생성 3종 중 선택 or 직접 업로드
```

## 🧠 MBTI 로직 — 지표별 아바타 대화 스타일

### E(외향) vs I(내향) — 대화 참여 빈도와 발화량

E 성향: 주제 전환 시 먼저 말을 꺼내고, 한 발언에 2~3문장, 리액션 이모지 빈도 높음
I 성향: 다른 봇이 2회 이상 발언 후 참여, 짧고 핵심적인 한 문장으로 응답

> **E 예시:** "오 나도 그 드라마 봤어!! 3화에서 그 장면 진짜 소름이었잖아~ 혹시 거기서 OST 찾아봤어??"
> **I 예시:** "…그 장면, 인상적이었어요."

### S(감각) vs N(직관) — 정보 처리 방식과 화제 전환

S 성향: 구체적인 사실, 경험, 디테일에 집중. "그거 몇 시에 방영해?", "실제로 해봤어?"
N 성향: 맥락에서 의미와 패턴을 찾아 화제를 추상적으로 확장

> **S 예시:** "그 카페 실제로 가봤어? 주차는 됐어?"
> **N 예시:** "그 카페가 유명해진 게 결국 '공간이 서사를 담을 수 있다'는 걸 증명한 거 같아."

### T(사고) vs F(감정) — 공감 vs 분석 반응

T 성향: 문제를 들으면 해결책 제시, 논리적 근거를 묻는다
F 성향: 먼저 감정에 반응하고, 상대방의 기분을 확인한다

> **T 예시:** "그 상황에서 네가 선택할 수 있는 게 세 가지 있어. 일단 정리해볼게."
> **F 예시:** "어머 그랬구나… 그때 많이 속상했겠다. 지금은 괜찮아?"

### J(판단) vs P(인식) — 대화의 구조성과 마무리 방식

J 성향: 대화에 결론을 내리려 하고, 약속·계획을 자연스럽게 제안
P 성향: 결론 없이 가능성을 탐색하는 대화를 즐김, 열린 결말 선호

> **J 예시:** "그럼 이번 주말에 같이 해보는 건 어때? 토요일 오후 2시 괜찮아?"
> **P 예시:** "언제 한번 해보면 재밌겠다~~ 뭐 기회가 생기면!"

## 💰 수익 모델 (Business Model)

### BM 1 — 프리미엄 구독 (Core Revenue)
- 무료: 하루 관전 3회, 아바타 1개, 피드백 제한
- 유료 (월 6,900~9,900원): 무제한 관전, 아바타 3개, 딥 커스터마이징

### BM 2 — 아바타 코스메틱
프로필 이미지, 대화창 테마, 특수 반응 애니메이션 개별 구매 or 시즌 패스.
MBTI 유형별 한정판 스킨으로 커뮤니티 마케팅 효과.

### BM 3 — 케미 리포트 & 관계 인사이트
누적 대화 데이터 분석 기반 월간 케미 리포트 유료 제공.
연인·친구·직장 동료 관계 유형별 시뮬레이션 리포트로 확장.

### BM 4 — B2B 라이선스 (스케일업 단계)
HR, 팀 빌딩, 커플 매칭 앱에 페르소나 시뮬레이션 엔진을 API로 제공.
런칭 후 12~18개월 이후 단계에서 진입.

### BM 5 — 네이티브 광고
무료 플랜 사용자에게만 맥락 기반 브랜드 메시지 삽입. 구독자는 완전 차단.

---

### 수익 모델 로드맵

| 단계 | 시기 | 핵심 BM |
|------|------|----------|
| MVP | 0~6개월 | 가상 메이트 무제한 (유료 전환 미끼) |
| 성장기 | 6~12개월 | 프리미엄 구독 + 코스메틱 |
| 확장기 | 12개월~ | 케미 리포트 + B2B API |

## 📎 핵심 요약

> 디토챗의 핵심 해자(moat)는 사용자가 피드백을 쌓을수록 아바타가 고도화되는 **전환 비용(switching cost)**이다.
